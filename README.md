#  Weather App

A simple and responsive Weather App built with **HTML**, **CSS**, and **JavaScript** that fetches real-time weather information using a weather API. Users can search for any city and instantly view its current weather details.

##  Features

*  Search weather by city name
*  Real-time temperature
*  Weather condition
*  Humidity
*  Wind speed
*  Invalid city error handling
*  Fully responsive design

##  Technology Used

* HTML5
* CSS3
* JavaScript (ES6)
* Weather API

##  Project Structure


Weather-App/
│── index.html

## 🌐 API Used

This project uses a weather API to fetch real-time weather information based on the city entered by the user.
